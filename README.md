# ofxHelios
Openframeworks addon for drawing vectors with the Helios Laser DAC

http://pages.bitlasers.com/helios/

Makefile builds for OSX and linux64

dependencies: libusb

cd example-colourpolyline<br>
make<br>
make RunRelease
